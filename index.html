金管楽器マウスピース断面ライブラリー

日本語

このサイトでは、金管楽器マウスピースの断面画像および 断面重ね合わせビューワーを公開しています。

  • PC版
    断面重ね合わせビューワー（日本語版・英語版）をご利用ください。
  • スマートフォン・タブレット
    PNG画像を閲覧できます。 対応する画像比較アプリを利用することで、 重ね合わせ比較も可能です。
  • 詳しい操作方法は出版物付属ガイドブックをご参照ください。

English

This website provides cross-sectional images of brass instrument mouthpieces together with an interactive overlay viewer.

  • For PC Users
    Use the Cross-Section Overlay Viewer (Japanese or English version).
  • For Smartphone / Tablet Users
    PNG images can be viewed directly. Overlay comparison is also possible using compatible image comparison applications.
  • For detailed operating instructions, please refer to the accompanying Guidebook.
